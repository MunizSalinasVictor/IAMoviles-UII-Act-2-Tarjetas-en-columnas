# IAMoviles-UII-Act-2-Tarjetas-en-columnas

### **Estructura Principal**
▸ Crear **5 cards** organizadas en **una sola columna**

▸ **Composición de cada card:**
- **Lado izquierdo**: Imagen cargada directamente desde internet
- **Lado derecho**: 
  - **Título** (parte superior)
  - **Descripción en negritas** (debajo del título)

▸ **Espaciado entre cards:** 10-15px
▸ *(Si el espacio es excesivo, reducir a 5px)*

---

## 🎨 **Esquema de Colores**

▸ **Fondo general:** Gris oscuro

▸ **AppBar:** Título "**Crunchyroll**"

▸ **Títulos:** Color naranja

▸ **Subtítulos:** Color naranja en **negritas**

▸ **Cards:** Fondo con **gradiente naranja**
